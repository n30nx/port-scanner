## port-scanner
A simple port scanner with rust

## Usage
Clone the repo
```sh
git clone github.com/n30nx/port-scanner
```

```sh
cd port-scanner
```

Run cargo build
```sh
cargo build --release
```

And you can start using the tool
```sh
target/release/port-scanner ip port_start,port_end scan_timeout
```
