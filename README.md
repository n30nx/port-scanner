## port-scanner
A simple port scanner with rust

## Usage
```sh
# Clone the repo
$ git clone github.com/n30nx/port-scanner

# Change directory to port-scanner
$ cd port-scanner

# Run cargo build
$ cargo build --release

# Start using the tool
$ target/release/port-scanner ip port_start,port_end scan_timeout
```
